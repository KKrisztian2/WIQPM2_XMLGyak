Ütemterv

Hét Elmélet Gyakorlat
1. Adatkezelés alapfogalmai, adatperzisztencia szintje, a file kezelés struktúrája és funkciói. C alapú filekezelés, Java alapú filekezelés; File szintű perzisztencia a C nyelvben
2. RDBMS alapfogalmai; DB fejlesztés lépései, SDM modellek áttekintése, ER adatmodell, az ER alapú modellezés lépései File szintű perzisztencia a Java nyelvben
3. ER adatmodellezés módszertana, az EER és UML adatmodellek. Relációs adatmodell, relációs struktúra és integritási feltételek. ER modell konverziója. Az ER tervezés gyakorlása
4. SQL nyelv áttekintése, SQL DDL nyelvi elemek. Szerver architektúra,Oracle APEX és Oracle SQLDeveloper felület. Az ER tervezés gyakorlása. A relációs modell szerkezeti elemek létrehozatala.
5. Relációs adatmodell műveleti része, relációs algebra. SQL DML és DQL alapok. Az ER tervezés gyakorlása. A relációs modell szerkezeti elemek létrehozatala.
6. SQL SELECT utasítás további elemei, kapcsolódás a relációs algebrához, al-SELECT és összetett lekérdezések. Relációs tervezés gyakorlása. Féléves feladat bemutatása
7. SQL speciáis elemei: dátum kezelés, sztring kezelés. Tranzakció elemei, VIEW mechanizmus. Fizikai szintű tárolási módszerek, indexelés SQL DML és DQL gyakorlása
8. Az adatmodellezés problémái, a függőségek szerepe. Függőségek szabályai, Normálformák. SQL DML és DQL gyakorlása
9. Dekompozíciós szabályok. Adatbázisok de-normalizálása, N1NF adatbázisok. Normalizálás, SQL gyakorlása
10. Az SQLite DBMS architektúra elemei, SQL-API rendszerek, SQL_API típusai; SQLite API C nyelvből, C programok fejlesztése Az SQLite C-API, SQL API elemeinek gyakorlása. Féléves feladat bemutatása
11. A JDBC keretrendszer architektúrája, JDBC - Java osztályok, alap funkciókat megvalósító Java JDBC programok fejlesztése. Az SQLite C-API, SQL API elemeinek gyakorlása
12. Adatbázisok védelmi modelljei, A DAC védelmi modell, szerepkörök alkalmazása; a MAC védelmi modell eleme Az JDBC Java elemeinek gyakorlása
13. Komplex adatbázis tervezési és implementációs feladat bemutatása. Kitekintés a NoSQL modelleke. Összefoglalás és konzultáció Féléves feladat bemutatása
